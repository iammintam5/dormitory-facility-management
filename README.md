# Quản lý cơ sở vật chất Ký túc xá PTIT HCM

## Phạm vi hệ thống
Hệ thống tập trung vào vòng đời tài sản: Nhập kho → Cấp phát → Sử dụng tại phòng → Báo hỏng → Bảo trì → Thu hồi hoặc tiếp tục sử dụng → Đề xuất thanh lý → Phê duyệt → Hoàn tất thanh lý.

> **Lưu ý:** Nghiệp vụ kiểm kê định kỳ không thuộc phạm vi triển khai của phiên bản hiện tại.
